README
------

This repository includes several small Kivy applications that 
demonstrate key features needed in the PRISM gui.  PRISM is a 
reactor simulator.

Project members:

J. Roberts, 
Jacob Hayhurst

Kivy API: http://kivy.org/docs/api-kivy.html
